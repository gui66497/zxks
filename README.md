### 在线考试系统
#### 项目介绍
项目主要有题库、试卷、考试、成绩、考生、统计等几大模块，题型支持单选、多选、判断、填空、简答等

#### 项目技术栈

前端：BootStrap、JQuery

后端：Jersey、Spring 、Mybatis
